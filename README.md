Na aula Python Dev, foi criado um site com Inteligência Artificial. proposta foi desenvolver um ChatBot interativo, onde o usuário pode escrever mensagens e receber respostas automáticas, 
simulando um bate-papo com um assistente virtual. 
A aplicação foi toda feita com a biblioteca Streamlit, que permite criar interfaces de sites diretamente com código Python.


Na aula Python Power UP, foi feito um projeto de automação para cadastro de produtos, preços em um sistema.
A ideia do projeto é entrar em um site (que foi criado especialmente para essa aula), fazer o login e o cadastro de todos os produtos de forma automática.


Na aula Python IA, foi feita uma análise de Score de clientes em um banco, em uma base de dados de 100.000 clientes, para verificar o score (pontuação) do cliente e assim saber se é um bom cliente ou não.
Inicialmente foi feito um tratamento na base de dados e criado alguns algoritmos de classificação para verificar qual deles é o melhor.
Com isso nós conseguimos fazer uma previsão dos clientes para saber se ele vai ser um cliente com um bom score ou não.
